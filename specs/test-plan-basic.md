# TODO・メモアプリ Basic粒度テスト計画

## Application Overview

TODO・メモ管理アプリケーションのbasic粒度テスト計画。ユーザー認証、TODO基本操作、メモ基本操作、ナビゲーション機能をカバーする5-10本の軽量テスト。正常系シナリオを中心とした基本的なユーザー操作フローを検証する。

## Test Scenarios

### 1. ログイン機能

**Seed:** `tests/seed.spec.ts`

#### 1.1. 正常ログイン

**File:** `tests/login/login-success.spec.ts`

**Steps:**
  1. ブラウザでhttp://localhost:5173にアクセス
  2. ログイン画面が表示されることを確認
  3. ユーザー名入力フィールドに有効なユーザー名を入力
  4. パスワード入力フィールドに有効なパスワードを入力
  5. 「ログイン」ボタンをクリック

**Expected Results:**
  - ログイン画面が正常に表示される
  - ユーザー名とパスワードが入力できる
  - ログイン成功後、ダッシュボードに遷移する
  - ナビゲーションメニューが表示される

#### 1.2. 無効な認証情報でのログイン失敗

**File:** `tests/login/login-failure.spec.ts`

**Steps:**
  1. ログイン画面にアクセス
  2. 無効なユーザー名を入力
  3. 無効なパスワードを入力
  4. 「ログイン」ボタンをクリック

**Expected Results:**
  - エラーメッセージが表示される
  - ログイン画面にとどまる
  - 入力内容がクリアされない

### 2. TODO機能

**Seed:** `tests/seed.spec.ts`

#### 2.1. TODO作成

**File:** `tests/todo/todo-create.spec.ts`

**Steps:**
  1. 有効な認証情報でログイン
  2. ナビゲーションメニューから「TODO」をクリック
  3. TODO一覧ページが表示されることを確認
  4. 新しいTODO入力フィールドにタスク名を入力（例：「報告書の作成」）
  5. Enterキーを押すまたは追加ボタンをクリック

**Expected Results:**
  - TODO一覧ページが表示される
  - 新しいタスクがTODO一覧に追加される
  - 追加したタスクが未完了状態で表示される

#### 2.2. TODO完了操作

**File:** `tests/todo/todo-complete.spec.ts`

**Steps:**
  1. 事前にTODOを1つ以上作成
  2. TODO一覧で未完了のタスクを確認
  3. タスクの「完了」ボタンをクリック

**Expected Results:**
  - タスクが完了状態に変更される
  - 完了したタスクが視覚的に区別される

#### 2.3. TODO削除

**File:** `tests/todo/todo-delete.spec.ts`

**Steps:**
  1. 事前にTODOを1つ以上作成
  2. TODO一覧で削除したいタスクを特定
  3. タスクの「削除」ボタンをクリック
  4. 確認ダイアログがある場合は確定

**Expected Results:**
  - タスクがTODO一覧から削除される
  - 削除確認が適切に動作する

### 3. メモ機能

**Seed:** `tests/seed.spec.ts`

#### 3.1. メモ作成

**File:** `tests/memo/memo-create.spec.ts`

**Steps:**
  1. 有効な認証情報でログイン
  2. ナビゲーションメニューから「メモ」をクリック
  3. メモページが表示されることを確認
  4. メモ入力欄にテキストを入力（例：「2025年1月12日の会議内容」）
  5. 保存ボタンをクリックまたは自動保存を確認

**Expected Results:**
  - メモページが表示される
  - 入力したメモが保存される
  - メモ一覧に新しいメモが表示される

#### 3.2. メモ一覧表示

**File:** `tests/memo/memo-list.spec.ts`

**Steps:**
  1. 事前にメモを複数作成
  2. メモページにアクセス
  3. 作成済みメモの一覧表示を確認

**Expected Results:**
  - 作成したメモがすべて一覧に表示される
  - メモの内容が正しく表示される

### 4. ナビゲーション機能

**Seed:** `tests/seed.spec.ts`

#### 4.1. 基本ナビゲーション

**File:** `tests/navigation/basic-navigation.spec.ts`

**Steps:**
  1. ログイン後、ダッシュボードを確認
  2. 「TODO」ボタンをクリックしてTODOページへ遷移
  3. 「メモ」ボタンをクリックしてメモページへ遷移
  4. 「トップ」ボタンをクリックしてダッシュボードへ戻る

**Expected Results:**
  - 各ナビゲーションリンクが正常に動作する
  - ページ遷移が適切に行われる
  - 現在のページが視覚的に識別できる

#### 4.2. ログアウト

**File:** `tests/navigation/logout.spec.ts`

**Steps:**
  1. ログイン済み状態を確認
  2. ナビゲーションメニューから「ログアウト」をクリック

**Expected Results:**
  - セッションが終了される
  - ログイン画面にリダイレクトされる
  - 認証が必要なページにアクセスできなくなる
